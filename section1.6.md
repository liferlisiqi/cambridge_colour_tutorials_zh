# Tutorials: white balance

[原文连接](https://www.cambridgeincolour.com/tutorials/white-balance.htm)  

白平衡（WB）是消除不真实偏色的过程，为了使在人眼呈现白色的对象在照片中也呈现白色。 正确的相机白平衡必须考虑到光源的“色温”，这是指白光的相对较暖或较冷。 我们的眼睛非常擅长判断不同光源下的白色，但是数码相机通常在自动白平衡（AWB）方面遇到很大的困难，并会产生难看的蓝色，橙色甚至绿色的偏色。 了解数字白平衡可以有助于避免这些偏色，从而在更广泛的照明条件下改善照片。

## BACKGROUND: COLOR TEMPERATURE

色温描述了“黑体”表面色温的光谱， 黑体是吸收所有入射光的对象，既不反射也不允许其通过。 在我们的日常经验中，大致可以理解黑体辐射是在加热金属或石头时：当它们达到一个温度时，它们会变成“红热”，然后在更高的温度下会“白热”。 类似地，处于不同色温的黑体也具有变化的“白光”色温。 尽管叫做“白光”，但其包含成分并不一定分布在可见光谱中：

![img](https://cdn.cambridgeincolour.com/images/tutorials/wb_bbdiagram.png)
![img](https://cdn.cambridgeincolour.com/images/tutorials/spectralsensitivity2.png)
![img](https://cdn.cambridgeincolour.com/images/tutorials/spectralsensitivity3.png)

可以看到 5000K 产生大致中性的光，而 3000K 和 9000K 分别移动以包含更多橙色和蓝色波长的光谱。 随着色温升高，颜色分布变冷。 这看似不直观，是由于较短的波长包含较高的能量。

既然摄影师不会与黑体打交道，为什么色温进行光线描述呢？ 是因为，日光和钨丝灯之类的光源紧密模仿了黑体产生的光源分布，尽管诸如荧光灯和大多数商业照明之类的其他光源远远偏离了黑体。 由于摄影师从未使用术语“色温”来指代真正的黑体光源，因此该术语隐含为与黑体颜色相似的对应色温。 下表是一些常见光源的相关色温的经验法则指南：

| **Color Temperature** | **Light Source**                       |
| --------------------- | -------------------------------------- |
| 1000-2000 K           | Candlelight                            |
| 2500-3500 K           | Tungsten Bulb (household variety)      |
| 3000-4000 K           | Sunrise/Sunset (clear sky)             |
| 4000-5000 K           | Fluorescent Lamps                      |
| 5000-5500 K           | Electronic Flash                       |
| 5000-6500 K           | Daylight with Clear Sky (sun overhead) |
| 6500-8000 K           | Moderately Overcast Sky                |
| 9000-10000 K          | Shade or Heavily Overcast Sky          |

## IN PRACTICE: JPEG & TIFF FILES

由于某些光源与黑体辐射器不同，因此白平衡除色温外还使用第二个变量：绿色-品红色偏移。 在普通日光下通常不需要调整绿色-品红色偏移，但是荧光灯和其他人工照明可能需要对白平衡进行大量的绿色-品红色调整。

幸运的是，大多数数码相机都包含各种预设的白平衡，因此您在关键拍摄期间不必处理色温和绿色-品红色偏移。 这些符号的常用符号在左侧列出。

前三个白平衡允许一定范围的色温， 自动白平衡在所有数码相机中均可用，并在有限的范围内使用最佳猜测算法-通常在3000/4000 K至7000 K之间。自定义白平衡可让您在相同照明下拍摄已知灰度参考的图像，并且 然后将其设置为以后照片的白平衡。 使用“开尔文”可以在较宽的范围内设置色温。

其余六个白平衡按色温升高的顺序列出，但是许多紧凑型相机不包括阴影白平衡。 某些相机还包括“荧光灯H”设置，该设置旨在用于较新的日光校准荧光灯。

上述白平衡的描述和符号只是对它们在最佳照明条件下实际照明的粗略估计。 实际上，根据时间、海拔或朦胧程度，也可以使用阴天代替日光。 通常，如果您的图像在LCD屏幕预览中显得太冷（无论设置如何），则可以通过在上面的列表中选择一个更小的符号来快速提高色温。 如果图像仍然太冷（或者朝另一个方向变暖），则可以采用开尔文设置手动输入温度。

如果所有其他方法都失败了，并且在计算机上检查图像仍然没有正确的白平衡，则可以调整色彩平衡以消除其他色偏。 另外，在Photoshop中使用“色阶”工具时，可以使用“设置灰点”滴管单击无色参考（请参阅中性参考部分）。 应避免使用这两种方法，因为它们会严重降低图像的位深度。

## IN PRACTICE: THE RAW FILE FORMAT

到目前为止，最好的白平衡解决方案是使用RAW文件格式（如果您的相机支持）拍摄照片，因为这些文件使您可以*在*拍照后设置白平衡。 RAW文件还允许您根据更宽的色温范围和绿色-品红色偏移设置白平衡。

用原始文件执行白平衡是快速简便的。 您可以调节温度和绿色-洋红色滑块，直到消除偏色为止，也可以单击图像中的中性参考（请参阅下一节）。 即使只有一张照片包含中性参考，您也可以单击它，然后将剩余的白平衡设置用于其余照片（假设使用相同的照明）。

## CUSTOM WHITE BALANCE: CHOOSING A NEUTRAL REFERENCE

中性参考通常用于对颜色要求严格的项目，或者用于预期自动白平衡会遇到问题的情况。 中性参考可以是场景的一部分（如果幸运的话），也可以是随身携带的便携式物品。 下面是一个在蓝蓝的夜景中幸运引用的示例。

## NOTES ON AUTO WHITE BALANCE

即使在正常的日光条件下，某些主题也会给数码相机的自动白平衡带来问题。 举个例子，由于独特的拍摄主题，图像是否已经过暖或过冷。 下图显示了主体是红色的情况，因此相机将其误认为是由暖光源引起的偏色。 然后相机会尝试对此进行补偿，以使图像的平均颜色更接近中性，但这样做会在不知不觉中使石头偏蓝色， 一些数码相机比其他数码相机更容易受到此影响。

当照片中至少包含一种白色或明亮的无色元素时，数码相机的自动白平衡通常会更有效。 当然，请勿尝试将组成更改为包括无色对象，但是缺少它可能会导致自动白平衡出现问题。 如果下图中没有白船，相机的自动白平衡错误地创建了色温稍高的图像。

## IN MIXED LIGHTING

具有不同色温的多种光源会使白平衡进一步复杂化。 某些照明情况甚至可能没有真正的“正确”白平衡，并且将取决于色彩精度最重要的位置。

<div align="center">
<img src="https://cdn.cambridgeincolour.com/images/tutorials/wb_mixed-ex2.jpg" height="500px" alt="Low Noise(Smooth Colorless Gray)" ><img src="https://cdn.cambridgeincolour.com/images/tutorials/wb_mixed-ex.jpg" height="500px" alt="High Noise(Patches of Color)" >
</div>

在混合照明下，自动白平衡通常会计算整个场景的平均色温，然后将其用作白平衡。 这种方法通常是可以接受的，但是与我们用眼睛观察到的情况相比，自动白平衡往往会夸大每个光源的色温差异。

在混合的室内和自然光线下，色温的夸大差异通常最为明显。 对于每个照明区域，关键图像甚至可能需要不同的白平衡。 另一方面，有些人可能更喜欢保持色温不变。

请注意左侧的建筑物色调很暖，而天空色调很冷。 这是因为白平衡是根据月光设置的-发出了下面人造照明的暖色温。 基于自然光的白平衡通常会产生更逼真的照片。 选择“石头”作为白平衡参考，看看天空是如何变得不现实的蓝色。
